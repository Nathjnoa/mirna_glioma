## Naming Conventions

- **Timestamps**: Most output files include `_YYYYMMDD_HHMMSS` suffix for run traceability.
- **Comparison tags**: DE outputs use the `analysis_id` from `config/de_specs.csv`.
- **Run tags**: Survival GSEA outputs use `--run_tag` (default: `SurvivalRank_CoxZ_miRPathDB`).
- **Export formats**: Publication figures exported as PDF (vector), SVG (editable), PNG (raster 600 dpi).

---

## Git Tracking

Only PNG figures in `results/figures/` are tracked in the repository (per `.gitignore`). All tables, DE results, logs, and PDF/SVG figures are local-only and must be regenerated from scripts.
